# Getting Started
1. Clone repo
2. npm i
3. npm run rebuild
4. npm start

### Build and Test
# Test
npm run test

# migrate db and seed admin
npm run rebuild
npm run migrate-rebuild

# seed data
npm run rebuild
npm run seed-employer
npm run seed-labor

# Start a server
npm start
